## 주요 환경 변수

```
DATABASE_URL      # PostgreSQL 연결 문자열
JWT_SECRET        # JWT 서명 키
PORT              # 서버 포트 (기본 3000)
FRONTEND_URL      # CORS 허용 출처 (콤마 구분으로 복수 지정 가능, 예: http://localhost:3001,http://192.168.x.x:3001)
ADMIN_EMAIL       # 초기 관리자 이메일
ADMIN_PASSWORD    # 초기 관리자 비밀번호
```
